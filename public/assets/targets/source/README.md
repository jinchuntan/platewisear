# Source target images

These are the curated food-waste images used to (a) compile the MindAR
`.mind` file and (b) preview exhibits in Demo Mode.

| Order (targetIndex) | File                  | What it shows                    |
|---------------------|-----------------------|----------------------------------|
| 0                   | `leftover-rice.jpg`   | A plate of leftover cooked rice  |
| 1                   | `fruit-peels.jpg`     | Citrus fruit peels               |
| 2                   | `bread-waste.jpg`     | A large pile of surplus bread    |
| 3                   | `mixed-leftovers.jpg` | Mixed vegetable scraps/trimmings |
| 4                   | `drink-waste.jpg`     | Discarded drinks & single-use cups |

Keep filenames stable — `src/food-targets.js` and the Demo previews reference
them by these paths. To compile the targets file, see the parent
[`../README.md`](../README.md).

Good targets are sharp, well-lit, feature-rich (lots of detail/texture) and not
too uniform — the five above work well.
