# Legacy Hiro marker (no longer used)

PlateNudge now uses **image-target scanning** of curated food-waste images
(MindAR), not the Hiro marker. See [`../targets/README.md`](../targets/README.md).

These files are kept only for reference / history:

- `hiro-marker.jpg`, `HIRO.jpg` — the old standard Hiro marker.
- `image1.jpg`, `image2.jpg`, `image3.jpg` — the original food-waste photos; copies now live
  (with descriptive names) in `../targets/source/`.
- `sdg12.png` — SDG 12 logo.

The current "Scan images" page (`marker.html`) shows the curated food-waste targets instead.
