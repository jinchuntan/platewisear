# Hiro Marker

The marker used by PlateWise AR is the standard **Hiro** marker from ARToolKit, built into AR.js via `preset="hiro"`.

The marker page (`marker.html`) draws the Hiro pattern on a canvas element. You can also download or print the official Hiro marker image from:

- AR.js documentation: https://ar-js-org.github.io/AR.js-Docs/
- ARToolKit Hiro marker: https://github.com/AR-js-org/AR.js/blob/master/data/images/hiro.png

Place a high-resolution Hiro marker image file in this folder (e.g. `hiro.png`) if you wish to use an image asset instead of the canvas-drawn version.
